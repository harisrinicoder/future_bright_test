Django Templates let you create dynamic HTML pages by combining HTML with Django Template Language (DTL).

ðŸ”¹ Using Variables

Output dynamic content by enclosing variables in double curly braces: 

<h1>Welcome, {{ user.username }}!</h1>

ðŸ”¹ For Loops

Loop through lists or querysets with the {% for %} tag:

{% raw %}
<ul>
  {% for item in items %}
    <li>{{ item.name }}</li>
  {% empty %}
    <li>No items found.</li>
  {% endfor %}
</ul>
{% endraw %}

ðŸ”¹ Conditional Statements

Control flow with {% if %}, {% elif %}, and {% else %}:

{% raw %}
{% if user.is_authenticated %}
  <p>Hello, {{ user.username }}!</p>
{% elif user.is_anonymous %}
  <p>Welcome, Guest!</p>
{% else %}
  <p>Please log in.</p>
{% endif %}
{% endraw %}

ðŸ”¹ Template Inheritance

Create base templates to reuse layouts and extend them:

{% raw %}

<html>
  <body>
    {% block content %}{% endblock %}
  </body>
</html>


{% extends "base.html" %}
{% block content %}
  <h1>Home Page</h1>
{% endblock %}
{% endraw %}

ðŸ”¹ Linking Static Files

Use Django's {% load static %} tag to link CSS, JS, or images stored in your static/ folder:

{% raw %}
{% load static %}
<link rel="stylesheet" href="{% static 'css/style.css' %}">
<img src="{% static 'images/logo.png' %}" alt="Logo">
{% endraw %}
