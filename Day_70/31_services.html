🛠️ Django Services App – Future Bright Institute

This app manages the list of services offered by your company, such as website development, SEO, digital marketing, etc.

📦 Step 1: Create the Services App

python manage.py startapp services

⚙️ Step 2: Register the App

# settings.py
INSTALLED_APPS = [
    ...
    'services',
]

🗂️ Step 3: Create the Service Model

# services/models.py
from django.db import models

class Service(models.Model):
    name = models.CharField(max_length=100)
    description = models.TextField()
    icon = models.CharField(max_length=100, blank=True)

    def __str__(self):
        return self.name

🧱 Step 4: Make Migrations

python manage.py makemigrations
python manage.py migrate

📋 Step 5: Add to Admin Panel

# services/admin.py
from django.contrib import admin
from .models import Service

admin.site.register(Service)

🔍 Step 6: Display Services on Web Page

# services/views.py
from django.shortcuts import render
from .models import Service

def service_list(request):
    services = Service.objects.all()
    return render(request, 'services/service_list.html', {'services': services})

🔗 Step 7: Set Up URLs

# services/urls.py
from django.urls import path
from . import views

urlpatterns = [
    path('', views.service_list, name='service_list'),
]
# Include in fbi_project/urls.py
path('services/', include('services.urls')),

🖼️ Step 8: Create Service List Template


<h1>Our Services</h1>
<ul>
  {% for service in services %}
    <li>
      <strong>{{ service.name }}</strong>
      <p>{{ service.description }}</p>
    </li>
  {% endfor %}
</ul>
