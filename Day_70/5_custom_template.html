🛠️ Customizing Django Admin Layout

You can override Django's default admin interface by customizing templates such as base_site.html. This is useful to change branding, add navigation, or insert your logo.

📁 Step 1: Create Folder Structure

project_root/
├── templates/
│   └── admin/
│       └── base_site.html

⚙️ Step 2: Update settings.py

# settings.py

TEMPLATES = [
    {
        "BACKEND": "django.template.backends.django.DjangoTemplates",
        'DIRS': [BASE_DIR / "templates"],  # For global overrides like admin
        'APP_DIRS': True,  # For app-specific templates
        "OPTIONS": {
            "context_processors": [
                "django.template.context_processors.request",
                "django.contrib.auth.context_processors.auth",
                "django.contrib.messages.context_processors.messages",
            ],
        },
    },
]

📝 Step 3: Override base_site.html

Create or copy the file templates/admin/base_site.html:

<!-- templates/admin/base_site.html -->
{% raw %}
{% extends "admin/base.html" %}

{% block title %}Future Bright Admin{% endblock %}

{% block branding %}
  <h1 id="site-name"><a href="/admin/">🌟 Future Bright Institute Admin Panel</a></h1>
{% endblock %}

{% block nav-global %}
  <p style="padding: 10px;">Welcome Admin! You can manage everything from here.</p>
{% endblock %}
{% endraw %}

🎨 Step 4: Add Your Logo or CSS

You can load static files as well:

{% raw %}
{% load static %}
<link rel="stylesheet" href="{% static 'css/custom_admin.css' %}">
<img src="{% static 'images/logo.png' %}" alt="Logo">
{% endraw %}

Optional: Create a Custom CSS File

project_root/
├── static/
│   └── css/
│       └── custom_admin.css

🧪 Step 5: Test It

  • Run the server: python manage.py runserver
  • Visit: http://127.0.0.1:8000/admin/
  • You should now see the custom title, header, or styling

📝 Optional Admin Templates You Can Override

  • base_site.html — Main layout (branding)
  • login.html — Custom admin login screen
  • index.html — Admin dashboard page
  • change_form.html — Form page for editing objects
Customizing the admin panel allows you to reflect your brand and provide a better user experience for internal staff.
