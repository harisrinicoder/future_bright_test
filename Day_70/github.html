🌐 Git & GitHub Setup Guide

Learn how to track your Django project with Git and connect it to GitHub.

🪜 Step 1: Initialize Git in Your Project

cd F:\fbi\future_bright
git init

📝 Step 2: Add and Commit All Project Files

git add .
git commit -m "Initial commit for Future Bright project"

☁️ Step 3: Create a GitHub Repository

Visit github.com/harisrinicoder → Click New → Name it (e.g., future_bright) → Create repo (no README needed).

🔗 Step 4: Connect Local Git to GitHub

git remote add origin https://github.com/harisrinicoder/future_bright.git
git branch -M master
git push -u origin master

📤 Push Changes to GitHub

git add .
git commit -m "Your message here"
git push

📥 Pull Updates from GitHub

git pull

🚫 Optional: Add .gitignore

# .gitignore file in root of project (Ignore Python stuff)
*.pyc
__pycache__/

# Ignore environment & DB
.env
db.sqlite3
*.log

# Ignore only collected static/media — not your development static files
/staticfiles/
/mediafiles/

✅ Final Structure

future_bright/
├── .git/
├── .gitignore
├── manage.py
├── db.sqlite3
├── courses/
├── blog/
├── contact/

Your project is now version-controlled and synced with GitHub! Keep using git add ., commit, and push regularly.
