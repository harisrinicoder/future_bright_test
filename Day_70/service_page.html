🛠️ Django Services Page – Future Bright Institute

Create a simple static Services page with Django views and templates.

📦 Step 1: Create the Services View

# pages/views.py (or your existing app's views)
from django.shortcuts import render

def services_view(request):
    return render(request, 'pages/service.html')

🔗 Step 2: Add the URL Pattern

# pages/urls.py
from django.urls import path
from . import views

urlpatterns = [
    path('service/', views.services_view, name='service'),
]

# project urls.py
from django.urls import path, include

urlpatterns = [
    # other paths
    path('', include('pages.urls')),  # or your app name
]
